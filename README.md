## Solar System

We use **C++/openGL, glut** library to draw a solar system contains planets, sun and stars.

--- 

Build and run in linux ubuntu

build
```
g++ main.cpp -o sul -lGL -lGLU -lglut
```
run 
```
./sul 
```
